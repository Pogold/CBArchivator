CB_Arhivator - простая утилита, которая переименовывает и складывает в архив файлы и папки в формате согласно ЦБ

---------------------------------------------------------------------------------------------
### Порядок выполнения

1. Выбор папки с архивом
2. Выбор даты архива
4. Запуск создания архива с запросом
